JS Modern: ES6+ có gì mới? Nâng cấp code JavaScript của bạn

ES6 là bản cập nhật lớn nhất trong lịch sử JavaScript. Bài viết này sẽ giới thiệu những tính năng cốt lõi và hay dùng nhất giúp code của bạn trở nên ngắn gọn, dễ đọc và mạnh mẽ hơn.
October 12, 2025 · Hữu Khang

Giới thiệu: ES6 là gì?#

ES6 (hay ECMAScript 2015) là phiên bản thứ sáu của chuẩn ECMAScript, thứ đặt ra các quy tắc cho ngôn ngữ JavaScript. Hãy coi nó như một bản “nâng cấp hệ điều hành” cực lớn cho JavaScript. Trước ES6, code JavaScript có phần dài dòng và đôi khi khó quản lý. ES6 ra đời đã thay đổi cuộc chơi, mang đến những cú pháp mới giúp lập trình viên viết code “sạch” và hiệu quả hơn.

Ngày nay, các tính năng của ES6+ (ES6 và các phiên bản sau nó) đã trở thành tiêu chuẩn. Hãy cùng điểm qua những thay đổi đáng giá nhất nhé!

1. Tạm biệt var, chào let và const#

Trước ES6, chúng ta chỉ có var để khai báo biến. var có một số hành vi khó hiểu liên quan đến “scope” (phạm vi). ES6 giới thiệu let và const để giải quyết vấn đề này.

  • let: Cho phép bạn khai báo các biến có phạm vi trong khối (block scope). Tức là biến chỉ tồn tại bên trong cặp dấu ngoặc nhọn {} gần nhất.
  • const: Giống như let, nhưng dùng để khai báo hằng số. Tức là giá trị của biến không thể được gán lại sau khi khai báo.

Ví dụ Code:#

// Trước ES6 (dùng var)
for (var i = 0; i < 3; i++) {
  console.log(i); // In ra 0, 1, 2
}
console.log(i); // Vẫn in ra 3! Biến i bị "rò rỉ" ra ngoài vòng lặp.

// Với ES6 (dùng let)
for (let j = 0; j < 3; j++) {
  console.log(j); // In ra 0, 1, 2
}

2. Arrow Functions (=>) - Viết hàm ngắn gọn#

Arrow function cung cấp một cú pháp ngắn hơn để viết các hàm, đặc biệt hữu ích cho các hàm callback.

Ví dụ Code:#

// Trước ES6
const numbers = [1, 2, 3];
const doubledOld = numbers.map(function(n) {
  return n * 2;
});
console.log(doubledOld); // [2, 4, 6]

// Với ES6
const doubledNew = numbers.map(n => n * 2); // Ngắn hơn rất nhiều!
console.log(doubledNew); // [2, 4, 6]

3. Template Literals (`) - Tạm biệt phép + chuỗi#

Template Literals cho phép bạn nhúng biến và biểu thức vào trong chuỗi một cách dễ dàng, cũng như tạo chuỗi nhiều dòng mà không cần \n.

Ví dụ Code:#

const name = "Khang";
const age = 22;

// Trước ES6
const greetingOld = "Xin chào, tôi là " + name + " và tôi " + age + " tuổi.";

// Với ES6
const greetingNew = `Xin chào, tôi là ${name} và tôi ${age} tuổi.`;

console.log(greetingNew);

4. Destructuring - “Giải nén” Object và Array#

Destructuring là một cú pháp tiện lợi cho phép bạn trích xuất dữ liệu từ Mảng (Array) hoặc Đối tượng (Object) và gán chúng vào các biến riêng biệt.

Ví dụ Code:#

// Destructuring cho Object
const person = {
  firstName: "Hữu",
  lastName: "Khang",
};
// Cách cũ: const firstName = person.firstName; const lastName = person.lastName;
const { firstName, lastName } = person; // Cách mới
console.log(firstName); // "Hữu"

// Destructuring cho Array
const colors = ["đỏ", "vàng", "xanh"];
// Cách cũ: const red = colors[0]; const yellow = colors[1];
const [red, yellow] = colors; // Cách mới
console.log(red); // "đỏ"

Kết luận#

Những tính năng trên chỉ là một phần nhỏ trong những gì ES6+ mang lại, nhưng chúng là những thay đổi có tác động lớn nhất đến cách chúng ta viết code JavaScript hàng ngày. Việc áp dụng các cú pháp hiện đại này không chỉ giúp code của bạn ngắn gọn hơn mà còn dễ đọc, dễ bảo trì và ít lỗi hơn. Hãy bắt đầu nâng cấp code của bạn ngay hôm nay!
