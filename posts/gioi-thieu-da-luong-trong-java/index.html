OOP: Giới thiệu về Đa luồng (Multithreading) trong Java

Tại sao phải chờ đợi? Multithreading cho phép chương trình Java của bạn thực hiện nhiều công việc cùng lúc, tăng tốc độ và khả năng phản hồi. Hãy cùng tìm hiểu hai cách cơ bản để tạo ra các luồng.
October 12, 2025 · Hữu Khang

Giới thiệu: Tại sao phải làm nhiều việc cùng lúc?#

Hãy tưởng tượng bạn là một đầu bếp trong một nhà hàng. Nếu bạn làm tuần tự: nấu xong món súp, rồi mới bắt đầu nhặt rau, rồi mới thái thịt… thì khách hàng sẽ phải chờ rất lâu. Một đầu bếp giỏi sẽ làm nhiều việc cùng lúc: bắc nồi súp lên hầm, trong lúc đó đi nhặt rau và thái thịt.

Đa luồng (Multithreading) trong lập trình cũng giống như vậy. Thay vì thực hiện các tác vụ một cách tuần tự, nó cho phép chương trình của bạn chia nhỏ công việc và thực hiện chúng song song trên các “luồng” (thread) khác nhau. Lợi ích chính là:

  • Tăng hiệu năng: Tận dụng tối đa sức mạnh của CPU đa nhân.
  • Tăng khả năng phản hồi: Giữ cho giao diện người dùng không bị “đơ” khi đang xử lý một tác vụ nặng ở phía sau.

Trong Java, có hai cách phổ biến để tạo ra một luồng.

1. Cách 1: Kế thừa từ lớp Thread#

Đây là cách tiếp cận đơn giản và trực tiếp nhất.

  • Bước 1: Tạo một lớp mới kế thừa (extends) từ lớp java.lang.Thread.
  • Bước 2: Ghi đè (override) phương thức run(). Toàn bộ công việc mà luồng sẽ thực hiện sẽ được đặt bên trong phương thức này.
  • Bước 3: Tạo một đối tượng từ lớp đó và gọi phương thức start() để bắt đầu thực thi luồng. (Lưu ý: gọi start(), không phải run()).

Ví dụ Code:#

// Tạo một luồng bằng cách kế thừa lớp Thread
class MyThread extends Thread {
    private String threadName;

    public MyThread(String name) {
        this.threadName = name;
    }

    // Công việc của luồng sẽ được viết ở đây
    @Override
    public void run() {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Luồng " + threadName + " đang đếm: " + i);
            try {
                // Tạm dừng 1 giây để giả lập công việc
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                System.out.println("Luồng " + threadName + " đã bị ngắt.");
            }
        }
        System.out.println(">> Luồng " + threadName + " đã hoàn thành.");
    }
}

public class ThreadExample {
    public static void main(String[] args) {
        // Luồng chính (main thread) bắt đầu
        System.out.println("Bắt đầu chương trình...");

        // Tạo và bắt đầu luồng thứ nhất
        MyThread thread1 = new MyThread("A");
        thread1.start(); // Bắt đầu thực thi luồng A

        // Tạo và bắt đầu luồng thứ hai
        MyThread thread2 = new MyThread("B");
        thread2.start(); // Bắt đầu thực thi luồng B

        // Luồng chính kết thúc
        System.out.println("Chương trình đã kết thúc.");
    }
}

2. Cách 2: Triển khai từ Runnable Interface#

Đây là cách được ưa chuộng hơn vì nó linh hoạt hơn. Java không cho phép đa kế thừa, nên nếu lớp của bạn đã kế thừa một lớp khác, bạn không thể kế thừa Thread được nữa. Runnable giải quyết vấn đề này.

  • Bước 1: Tạo một lớp mới triển khai (implements) java.lang.Runnable interface.
  • Bước 2: Tương tự, triển khai phương thức run() và đặt công việc của luồng vào đó.
  • Bước 3: Tạo một đối tượng từ lớp đó.
  • Bước 4: Tạo một đối tượng Thread mới và truyền đối tượng Runnable của bạn vào hàm khởi tạo của nó.
  • Bước 5: Gọi phương thức start() của đối tượng Thread..

Ví dụ Code:#

// Tạo một luồng bằng cách triển khai Runnable
class MyRunnable implements Runnable {
    private String threadName;

    public MyRunnable(String name) {
        this.threadName = name;
    }

    // Công việc của luồng
    @Override
    public void run() {
        for (int i = 1; i <= 5; i++) {
            System.out.println("Runnable " + threadName + " đang đếm: " + i);
            try {
                Thread.sleep(1000);
            } catch (InterruptedException e) {
                System.out.println("Runnable " + threadName + " đã bị ngắt.");
            }
        }
        System.out.println(">> Runnable " + threadName + " đã hoàn thành.");
    }
}

public class RunnableExample {
    public static void main(String[] args) {
        System.out.println("Bắt đầu chương trình...");

        // Tạo đối tượng Runnable
        MyRunnable runnable1 = new MyRunnable("C");
        // Bọc nó trong một đối tượng Thread để chạy
        Thread thread1 = new Thread(runnable1);
        thread1.start();

        MyRunnable runnable2 = new MyRunnable("D");
        Thread thread2 = new Thread(runnable2);
        thread2.start();
        
        System.out.println("Chương trình đã kết thúc.");
    }
}

Kết luận#

Đa luồng là một khái niệm mạnh mẽ trong Java. Bằng cách sử dụng Thread hoặc Runnable, bạn có thể tạo ra các ứng dụng có khả năng thực thi song song, giúp cải thiện đáng kể hiệu suất và trải nghiệm người dùng. Mặc dù cách tiếp cận ban đầu khá đơn giản, thế giới đa luồng còn ẩn chứa nhiều khái niệm phức tạp hơn như đồng bộ hóa (synchronization) và khóa (locking), nhưng hai phương pháp trên là bước khởi đầu vững chắc nhất cho bất kỳ ai muốn chinh phục chủ đề này.
